SECTION_03 / PHYSICAL SPECIFICATIONS

Everything about the Hardware

PicoBox housing design optimization for Arduino Uno Q architectures.

[ Core Logic ] A_UNO_Q

The Brain

Arduino UNO Q 4GB

A hybrid architecture segmenting physical/audio signals to prevent playback jitter while processing intensive local offline inference blocks.

  • CPU: Qualcomm QRB2210 (quad-core 2.0GHz)
  • RAM: 4GB LPDDR4
  • Storage: 32GB eMMC
  • OS: Debian Linux (dual-architecture)
[ Input Sense ] RESP_XVF

The Ears

ReSpeaker XVF3800

XMOS hardware layer far-field tracking allows clear, cross-room vocal commands under complex acoustic echo environments.

  • Mics: 4 MEMS digital microphones
  • Range: Up to 5 meters (far-field)
  • Coverage: 360° voice tracking
  • Features: Beamforming, AEC, Noise Supression
[ Output Engine ] USB_SPK_2.0

The Mouth

USB 2.0 Speaker

Delivers clean, natural synthesized speech. Perfect acoustics matching the inner space layout of PicoBox chassis.

  • Output: 2 x 3W drivers
  • Interface: USB + 3.5mm AUX
  • Alternative: JBL GO 3 (Prototyping)
  • Mount: Custom modular internal plate

04.2 Command Router Logic

router.py Python 3.11 
def route_command(transcribed_text):
    text = transcribed_text.lower()
    
    if "search and think" in text:
        # Mode: Research & Synthesis
        return call_deepseek(query, context, mode="synthesis")
        
    elif "search" in text:
        # Mode: Factual Retrieval
        return call_deepseek(query, context, mode="summarize")
        
    elif "think" in text:
        # Mode: Creative/Reasoning
        return call_deepseek(query, mode="creative")
        
    else:
        # Default: Conversational
        return call_deepseek(text, mode="conversational")
